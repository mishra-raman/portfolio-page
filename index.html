Adaptive Dynamic Range Optimization

This project implements the ADRO (Adaptive Dynamic Range Optimization) algorithm in a real-time smartphone app. The app processes audio input by dividing the signal into 32 frequency bands and amplifying each band based on gain values from a Gain Computation module. These gains are adjusted using estimated sound pressure level (SPL) percentiles, resulting in dynamic range adjustment of the audio signal.
